# Backend de l'application IDE

## Description
Cette partie de l'application est développé sous la forme d'une API service, qui sera accessible via des enpoints coté client. 
Les outils utilisés sont:
- Spring Boot (framework de Java)
- MySql (SGBDR)
### Pour lancer l'application
- D'abord, apès téléchargement du code source, il faut créer une base donnée, après quoi, on va dans le fichier application.properties,
  ajouter les différentes informations, nom de la base, username, mot de passe et le port.
- Lancer MySQL
- Positionner dans le fichier BackendApplication.java et lancer le projet
- Et la base URL est http://localhost:8080/api
- Pour trouver les différent endpoints, visiter les différents controllers de l'application
